Advertising Space

A very early applet that we developed, Adspace won an honourable mention in the first Java programming contest.

It cycles through a set of images just like a football stadium advertising hoarding. Clicking the mouse over the applet takes the browser to the URL link associated with the currently displayed image.

Get the class and config. files:

  • adSpace.class
  • Advert.class
  • adlist..txt

Feel free to download these files.
