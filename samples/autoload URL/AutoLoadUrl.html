AutoLoadUrl

The AutoLoadUrl applet load an URL automatically as soon as an java enabled browser is used. I use this from my home page to load an alternate Java-enabled page, if running with an Java-enabled browser.

Typical usage is as follows: 

<APPLET CODE="AutoLoadUrl.class" WIDTH=400 HEIGHT=100>
<PARAM NAME=url VALUE="http://www.algonet.se/~jannea/javahome.html">
<BLOCKQUOTE>
<HR>
If you were using a Java-enabled browser, a new java-enabled
page would be automatically loaded...
<HR>
</BLOCKQUOTE>
</APPLET>
