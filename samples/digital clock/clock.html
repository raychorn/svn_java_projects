This digital clock ver 1.1 is flicker free. It is a 7 segment BCD-LED clock. The only configurable part of the clock at this time is the color of the digital clock. User may choose his or her own foreground and border color. The background color is not configurable since I found that black is the best color for this digital clock background. Here is some comments people have left about this applet. FAQ for this applet.

Parameter

The parameters available for this digital clock are fcolor and border. Fcolor is the foreground color and border is the border color.

Usage

To use this applet in your page, follow these insructions

  1. Download the digital clock now.
  2. Put the class file into your homepage directory and change its mode to world readable.
    For instance, in unix prompt type: chmod 644 showtime.class
  3. Type in the following tag into your page file
    <applet code = "showtime.class" width=127 height=47>
  4. The default foreground color is green while the border color is brown. To change the colors, type the following tags
    <param name="fcolor" value="red,green,blue">
     <param name="border" value="red,green,blue">
     where the value of red, green and blue is between 0 and 255
  5. Finally, you need the closing tag
    </applet>

See the sample page.
