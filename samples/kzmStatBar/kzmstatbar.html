KzmStatBar

    Date: 28.Dec.95

    I wrote this applet for testing purpose, in order to give the opportunity to have a simple and easy to use statistic. There is a file, containing values, labels and colors of all the bars, and then display them basing on the dimension of the applet. Actually this is a "beta" release, i'll try to implement as soon as possible.

    PARAMETERS

    StatFile: (String, default=stat.txt)
    This is the name of the file contaning the informations about the chart.
    Here is an example of the file:
    10,BAR 1,red
    13,BAR 2,white
    15,BAR 3,green
    12,BAR 4,blue
    18,BAR 5,cyan
    11,BAR 6,yellow
    Copyright: (String)
