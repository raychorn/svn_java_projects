ColorMap applet

The ColorMap applet is a *NEW* style ImageMap.

  • Map images the EASY way!

  • Easy to use graphical method of creating image maps.

  • If you can draw an image, you can use it! Simple as that!
    • One Image is displayed to the user
    • One Image holds the map of active areas
    • ie. {Color equals Area}

  • Don't be a square! Use irregular areas in image maps

  • Disjoint and split areas(Yeah...spread that load, man!)

  • This SHAREWARE Applet allows multiple events per area :
    • Named areas - Display text when entered
    • Sound Effect areas - Play a sound when entered
    • Link URL areas - Link to other Web Sites when clicked
    • Clickable areas - Tell you where their coordinate and color when entered
    • Anything (Create your own ColorMappedArea class)!

  • Up to 16 million different areas per image

  • Plug it straight into your applets/applications NOW!
