- Import a CinnaMoney class


Importing CinnaMoney class is easily done with a single line of code such as the one following this:

import CinnaMoney;

- Make a simple example of a program using a CinnaMoney class?


The following is a simple program which creates a couple of CinnaMoney Objects using String Objects containing numeric values. It then adds those two CinnaMoney Objects together to created a third CinnaMoney value containing the results. It then prints their results. 

/**
 * Copyright (c) 1995, 1996 THOUGHT Inc. All Rights Reserved.

 *
 */

import java.util.*;
import java.lang.*;
import java.lang.*;
import CinnaMoney;

public class TestCinnaMoney1 {
	public static void main(String args[]) {
	String strn1 = new String("12000.00");
	CinnaMoney m1 = new CinnaMoney(strn1);
	strn1 = new String("-1200.00");
	CinnaMoney m2 = new CinnaMoney(strn2);
	CinnaMoney m3;
	m3 = m1.add(m2);
	System.out.print(m1.toString());
	System.out.print(" + ");
	System.out.print(m2.toString());
	System.out.print(" = ");
	System.out.println(m3.toString(2));  // this sets print precision to 2
	}
}

//=========================

Here's how to compile
javac TestCinnaMoney1.java
Java TestCinnaMoney1

This should print:
12000.00 + -1200.00 = 10800.00

- How do I compile a Java program/applet including CinnaMoney


Compiling a program to work with CinnaMoney is the same as for any other Java program. Just use the javac compiler to compile your source.
An example compile for a file called TestCinnaMoney.java would be:

javac TestCinnaMoney.java

- How do I execute a program/applet which was compiled to include CinnaMoney


Applet execution is documented in Sun's documentation for the Java Developer's Kit. CinnaMoney is included in the same fashion that any other java class is. As long as your CLASSPATH variable is set properly to include CinnaMoney, the CinnaMoney class should be found properly.

- How do I convert another datatype to a CinnaMoney Class?


Money can currently convert the following datatypes (int, long, float and String) to type CinnaMoney. The constructor is used to convert, and the following lines of code demonstrate using each. 
// Native datatypes
String st = new String("12.3");
long lv = 2334433;
int iv = 123;
float fv = 123.4;
// Now convert
CinnaMoney cm = new CinnaMoney(st);
CinnaMoney cm2 = new CinnaMoney(lv);
CinnaMoney cm3 = new CinnaMoney(iv);
CinnaMoney cm4 = new CinnaMoney(fv);

- Set the precision of a CinnaMoney Class?


Precision can be set in two ways, one way shifts the decimal (used internally) mainly, the other modifies the physical precision - truncating or adding zeros if necessary.

Shift decimal place to modify precision
aCinnaMoney.setPrecision(4);

Change precision
aCinnaMoney.changePrecision(4);

- Print a CinnaMoney Class as a String?


This is a trivial operation, and will default to the internal precision of the Object being printed.

aCinnaMoney.toString();

- Print a CinnaMoney class as a String using a specific printing precision?


This is a trivial operation, and will use the integer value passed into toString as the printing precision of the Object being printed.

aCinnaMoney.toString(2); // This sets precision to 2 decimal places
