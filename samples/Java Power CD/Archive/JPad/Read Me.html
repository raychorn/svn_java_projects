JPad 1.0 - Read Me!

Welcome to JPad™. JPad lets you:
  • Edit Java™ programs
  • Compile and run Java applications and applets
  • Create C bindings for building native methods
  • Disassemble Java classes
  • Locate Java syntax errors by just clicking on the error message

This release is available for Windows ® 95 and Windows NT 3.51.

Send comments, suggestions or bug reports to Chet Murphy at cmurphy@modelworks.com. Check out http://www.modelworks.com/express for information about other ModelWorks products including information about JPad Pro™.

Installation

There is no installer for this release. To install Express copy the files to a folder.

To setup JPad you will need to tell JPad where your Java directory is located. To do this open JPad and choose the Java Directory item from the Java menu and set the location of your Java directory. You will also need to tell Java where your Java package sources are located. You can do this by choosing the Package Directories item from the Java menu. For all the Java tools to work, one of the entries in the Package Directories must be "...\java\lib\classes.zip". JPad assumes that your Java directory in located your JPad directory and that the Java tools are located in "...\java\bin\"..

Read the JPad FAQ for more information.

Change History

JPad 1.0.5 (MWJ105.zip)

1. Fixed a bug that caused a system error when trying to run a Java application or when trying to disassemble a Java class with numeric constants in the class file.

2. Added a stop command to kill run-away Java applications and applets.

3. Added a line number indicator to the status bar.

4. Added tool bar buttons for compile, run and stop commands

JPad 1.0.4 (MWJ104.zip)

1. Made minor changes to output messages.

2. Updated the Run Applet command so that it sets the current directory before invoking JDK's appletviewer. This change lets the appletviewer run an applet correctly if the current directory has been changed from your applet directory. See the JPad FAQ for more information.

3. Removed unsupported menu items.

JPad 1.0.3 (MWJ103.zip)

1. Added the "Run Applet..." command.

2. Fixed the bug that prevented saving the tab setting.

3. Fixed the bug that caused an assertion failure when JPad's window was made "too" small.

4. Removed the '-l' option from the Disassemble menu command.

JPad 1.0.2 (MWJ102.zip)

1. Made changes to allow spaces in Java directory path names.

2. Added the "Disassemble" command.

JPad 1.0.1 (MWJ101.zip)

1. Fixed the bug that prevented the capture of output from the Java tools under Windows NT

2. Fixed the bug that prevented locating errors by clicking on the error message in the output window under Windows NT

3. Fixed the bug that prevented the output window from being cleared under Windows NT.

JPad 1.0 (MWJ10.zip)

1 Initial public release.
