Install distribution:

These instructions presume that you have already installed the Java Developer's Kit (Nutmeg was built with FCS1.0 of the JDK, if you don't already have it, you should download it now.) in some directory that will be referred to through the rest of this document as 

Sun Solaris:
cd <java home directory>
uncompress -c nutmeg-fcs1.01.tar.Z | tar xpfv -

(Warning, dos versions of pkunzip will truncate filenames rendering this package useless - you need an upgraded version of unzip which will support long filenames Winzip 6.0 if you don't already have it is a great package) 

Windows:

cd <java home directory>
pkunzip -d nutmeg-fcs1.01.zip

Using the package:

A simple Test program is included in this distribution using String Objects just to help you to see if you installed the program properly: 

// TestNutmeg.java
// Test Program to test if Nutmeg Installed

import java.io.*;
import java.util.*;
import java.lang.*;
import java.Nutmeg.*;

public class TestNutmeg {
	public static void main(String args[]) {
	Collection cltn = new Collection();
	String strn = new String("Hello");
	cltn.add(strn);
	strn = new String("World");
	cltn.add(strn);
	System.out.println(cltn.at(0));
	System.out.println(cltn.at(1));
	}
}
To see if This compiles correctly do the following - it is presumed that your CLASSPATH contains a directory reference to "." in it's list of directories. If this is not the case, then the java runtime might have trouble finding this test program when you try to run it with the Java package. 
cd /Nutmeg
javac TestNutmeg.java
Java TestNutmeg

This should print:
Hello
World

You have now successfully installed Nutmeg
