MRGBs Class

Class name:

MRGBs
Abstract:

Draw moving squares.
Parameter Name:

SEQUENCE
Parameter Value: Color sequence to be used for changing colors of the oval

Colors are coded as following:

  • R - red
  • O - orange
  • Y - yellow
  • G - green
  • B - blue
  • M - magenta
  • K - black
  • W - white


Example1:

<APPLET code="MRGBs.class" width=90 height=30> <param name=sequence value="RGY"> </APPLET>

Example2:

<APPLET code="MRGBs.class" width=400 height=20> <param name=sequence value="GWGWGWGW"> </APPLET>
