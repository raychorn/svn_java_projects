Caution: This page is only useful, if you use the Netscape 2.0 browser.

If you don't have this browser, use HotJava to look at the alpha version of JDance.

Some remarks:

Please be patient, if you start the applet the first time. There are some pictures/sounds to load, which can take some time.

Choose dance, figur and music and press "show dance". With "play" you can view the dance or with "step" you can examine it step by step.
