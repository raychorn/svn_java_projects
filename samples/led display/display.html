LED display applet was based on my senior project when I was in the University of Texas. My senior project was to build a micro-controller based LED display. The display can show at least 12 characters at a time and can hold up to 64 characters in total. Since Java has become a very hot language, I decided to transform my old project to a new and interactive applet

LED display applet can show up to 16 characters at a time and can hold data up to 20 messages. The user can also configure the scrolling directions for the messages and colors for each messages. Parameters

There are three parameters available for this applet.

messages
This applet can hold up to 20 messages.
; character is used to separate each messages.
<param name="messages" value="Welcome to;Benny's World">

msgcolor
This patameter is used to control the color of the messages. Total number of msgcolor has to be equal to the total number of the messages.
The syntax:
<param name="msgcolor" value="(red,green,blue),(red,green,blue)">
Where the value of red, green, green is between 0 to 255.

control
This parameter is used to control the direction of the scrolling messages. Each message can be controlled twice.
Follow this syntax:
<param name="control" value="0,1,2,0,3,4">

The controls are:
0: New message
1: Scroll the messages from right to left
2: Clear the display by entering space from the right
3: Scroll the messages from left to right
4: Clear the display by entering space from the left
5: Divide the display into two parts and each part will scroll to opposite direction
6: Clear the display by entering space from the bottom
7: Clear the display by entering space from the top
8: Scroll the messages upward
9: Scroll the messages downward

Note: Always put 0 for new messages.

Installation

To install the LED applet, follow this instructions:

  1. Download the LED applet now.
  2. Put the class file into your homepage directory and change its permission to world readable
    For instance, in Unix, the command is: chmod 644 gambar.class
  3. Type in the following tag into your page file
    <applet code = "gambar.class" width=408 height=44>
  4. Type in the parameters.
    <param name="messages" value="Enter your desired messages here">
    <param name="msgcolor" value="Enter your desired messages' colors here">
    <param name="control" value="Enter your desired messages' controls here">
  5. Finally, you need the closing tag
    </applet>
