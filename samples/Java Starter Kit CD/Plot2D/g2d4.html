Example 4

This is a replot of Example 2 with the ability to change the range of the plot by dragging the mouse.

Press a mouse button over the plot and drag out the new range rectangle. When the mouse button is released the plot is redrawn with the new range.

By pressing and releasing the mouse button with the shift key down the data is redrawn with the default range.
