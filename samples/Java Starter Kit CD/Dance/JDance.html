Tanzen lernen leicht gemacht

Sorry!

Mit der neuen Netscape beta 2 Version funktioniert das Applet nicht mehr so richtig. Ich arbeite daran, bin aber nahe daran die Schuld NetScape oder Sun zu geben.

Ich habe nun herausgefunden, dass NetScape 2.0b2 die Klasse Tango nicht laden moechte. Ich habe das Applet nun so abgeaendert, dass alles bis auf den Tango funktionieren sollte.

Aufgrund der Änderungen funktioniert im Moment leider nur die englische Version des Applets.

(9. Nov. 1995, Georg Heßmann)

Achtung: Diese Seite macht nur Sinn, wenn sie mit dem Browser Netscape 2.0 angezeigt wird!

Wenn Sie nur HotJava (alpha) zur Verfügung haben, dann können Sie die Alpha Version des Applets hier betrachten.

Bemerkung:

Bitte seien Sie etwas geduldig, wenn Sie das Applet das erste mal starten. Es müssen erst einige Bilder und Sounds übertragen werden, was schon ein wenig Zeit benötigen kann.

Wählen Sie einen Tanz, dazu eine Figur und eine Musik aus und drücken Sie dann "Tanz anzeigen". Danch können Sie die Figur mit "Animation" ansehen, bzw. mit "vorwärts" jeden einzelnen Schritt abgehen.

Informationen zu dem Java-Program (Source + Docu) gibt es hier.
