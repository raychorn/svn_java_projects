Client Side Image Map Authoring

A system has been written to allow users to edit the 'hot spots' on an image without leaving their Web browser. This system is written in JAVA, and requires a Web browser which supports the JAVA programming extensions. These browsers are to date Netscape 2.0b4 and 2.0b5. The system may work with other browers, but has not been tested.

The system is in an early stage of development, and has the following known problems :

  1. Hot spot areas seem to be a bit misplaced - only about half the area works properly. This now seems to be a bug in Netscape 2.0b4.
  2. There is no way to save or load the resultant map yet - this is being worked on.
  3. The applet does not clean up properly after itself
  4. Netscape 2.0b5 sometimes seems to crash on this applet.

Please note that this system is ALPHA development, and is unsupported. It is released in the hope that it might be useful, and does not have any warranty at all.
