How you can do this:

Adding Java rules and bullets to your HTML documents is not a complex task:

  1. Read this agreement
  2. Download J-tools package
  3. Unzip it
  4. Copy *.class files to a directory accessible from your WWW server
  5. Add calls to Java applets to your pages
  6. (optional) Send E-mail to integris@crl.com with subject line "JT-user" to receive information about J-tools new releases and updates

More Examples:

This is an example of a call to Java program which displays rainbow-like text

<applet code=" RnbText.class " width=200 height=50> <param name=text value="Integris"> </applet>

In this example we assume that applet code (RnbText.class file) is in the same directory as the HTML document page. We specify window size and text to be shown.

To customize appearance of Java applets you may want to change size or sequence of colors.

Change of the size is trivial: you just change the value of HEIGHT and WIDTH parameters.

For example, <applet code="SinText.class" width=150 height=40> will display letters in 150x40 size window.

To change color sequence you will need to change "sequence" parameter.

Colors are coded as following:

  • R - red
  • O - orange
  • Y - yellow
  • G - green
  • B - blue
  • M - magenta
  • K - black
  • W - white

So for white-black bullet use: <param name=sequence value="KW">

and for traffic light-like bullet use: <param name=sequence value="YRG">
